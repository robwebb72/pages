This is a journey into using VIM

MultiView

The commands for multi views are:-

action keystroke/seq
new split window (horizontal split) \ + w , s
new split window (vertical split) \ + w , v
close current split window (1) \ + w , q
move focus to window to the left \ + w , h
move focus to window to the right \ + w , l
move focus to window below \ + w , j
move focus to window above \ + w , k

(1) also closes the main window

Commands

:e \<filename>    - open file
:Ex \<directory> - browse directory to find file (if no directory given, will browse current dir)

also works with [ and {

vi keystrokes

yy - copies the current line
P - pastes before this line.  (p - paste after this line)
o - opens new line below cursor
O - opens new line above cursor
u - undo

Other keystrokes

diw - delete inner word
di" - delete everything between surrounding quote marks - also works with [ { " ' (
di( - delete everything between surrounding brackets
ciw - change inner word
ci( - change contents of parantheses
D - delete to end of line
dd - delete line
