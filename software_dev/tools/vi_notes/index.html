VIM

MultiView

The commands for multi views are:-

action keystroke/seq
new split window (horizontal split) [CTRL] + w , s
new split window (vertical split) [CTRL] + w , v
close current split window (1) [CTRL] + w , q
move focus to window to the left [CTRL] + w , h
move focus to window to the right [CTRL] + w , l
move focus to window below [CTRL] + w , j
move focus to window above [CTRL] + w , k

(1) also closes the main window

Commands

command action
:e \ open file
:Ex \ browse directory to find file (if no directory given, will browse current dir)

Inserting and Undoing

key action
o opens new line below cursor
O opens new line above cursor
u undo

Copying and Pasting

key action
yy copies (yanks) the current line
P paste before current line
p paste after current line

Deleting and Changing Text

key action
diw delete inner word
di" delete everything between surrounding quote marks - also works with [ { " ' (
ciw change inner word
ci( change contents of parantheses
D delete to end of line
dd delete line
